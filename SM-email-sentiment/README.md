# Overview

Title: Email Sentiment
Description: A Flow engine that accepts incoming csv & text files webhooked to an HTTP Adapter endpoint sent from an email provider of your choice (eg Mandrill, Sendgrid) and process with a sentiment analysis engine and (1) write the results to a TDO, (2) write the results back to a callbackUrl field if provided by the sender
